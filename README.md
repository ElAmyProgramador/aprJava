# aprJava
Como en la descripción de este repo, aquí subiré las practicas que haga mientras sigo el curso de [TodoCode](https://www.youtube.com/watch?v=qxXcI56NfnE&t=1936s).
¿Por qué hacer ejercicios tan simples como los del "Hola mundo", variables, flujos de control, etcétera?
Bueno, la razón es mas para poder acostumbrarme al IDE y como maneja las cosas, suelo usar los compiladore, editores de texto y todo desde la temrinal; ya estoy desacostumbrado a los IDE.
